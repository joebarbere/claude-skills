# geohot

A minimal Bear Blog theme inspired by [geohot.github.io/blog](https://geohot.github.io/blog/) (Jekyll Minima).

## Characteristics

- System font stack (no web fonts)
- Light-weight headings (font-weight: 400)
- Near-black text (`#111`) on near-white background (`#fdfdfd`)
- Blue links (`#2a7ae2`)
- 740px content width
- Light lavender code blocks (`#eef`)
- 5px dark gray top border accent on nav
- Minimal decoration throughout
