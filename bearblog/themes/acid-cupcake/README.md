# acid-cupcake

A dark Bear Blog theme inspired by the [Acid Cupcake Colour Scheme](https://github.com/jsantell/AcidCupcakeColourScheme).

## Characteristics

- Dark background (`#212426`) with warm cream text (`#FFFDEB`)
- Monospace font stack (SF Mono, Fira Code, Menlo, Consolas)
- Hot pink headings and accents (`#FF0077`)
- Cyan links (`#04DBE5`)
- Acid green code blocks, emphasis, and hover states (`#AAEE22`)
- Orange bold text and table headers (`#FFB412`)
- Grey comments and meta text (`#727879`)
- Pink text selection highlight
- Styled scrollbars (Webkit)

## Palette

| Color   | Hex       | Usage                           |
|---------|-----------|---------------------------------|
| Black   | `#212426` | Background                      |
| Cream   | `#FFFDEB` | Body text                       |
| Pink    | `#FF0077` | Headings, nav, accents          |
| Cyan    | `#04DBE5` | Links, h4                       |
| Green   | `#AAEE22` | Code, hover states, emphasis    |
| Orange  | `#FFB412` | Bold text, h3, table headers    |
| Grey    | `#727879` | Meta text, blockquotes, comments|
| Red     | `#f63d4e` | Errors                          |
| Yellow  | `#f6c83d` | Highlights                      |
